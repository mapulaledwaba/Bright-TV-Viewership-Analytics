📌 Overview
This project analyzes BrightTV’s viewership data to understand who watches, when they watch, and what content drives engagement. 

The goal is to uncover viewing patterns and provide strategic recommendations to help BrightTV grow its user base.

🛠️ Tools Used

Databricks — data import, cleaning, SQL aggregations

Excel — grouped data processing, pivot tables, charts

Miro — workflow mapping and project planning

Canva — presentation design and slide structuring

🔄 Process
Data Collection — imported BrightTV dataset into Databricks

Data Cleaning — fixed missing values, corrected formats, created grouped fields

Data Processing — SQL + Excel for demographics, channels, provinces, time patterns

Analysis — identified trends in behaviour and engagement

Visualization — built charts for demographics, channels, and time patterns

Insights & Recommendations — highlighted peak hours, top channels, key segments, and growth opportunities
